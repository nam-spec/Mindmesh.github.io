Binary Search Algorithm

Explanation

Binary search is a divide-and-conquer algorithm that finds the position of a target value within a sorted array. It works by repeatedly dividing the search interval in half.

Steps:

  • Start with the middle element of the array.
  • If the target value is less than the middle element, search the left half.
  • If the target value is greater, search the right half.
  • Repeat the process until the target value is found or the array is exhausted.

Visualization

Code

      
        function binarySearch(arr, target) {
          let left = 0;
          let right = arr.length - 1;
          while (left <= right) {
            let mid = Math.floor((left + right) / 2);
            if (arr[mid] === target) return mid;
            if (arr[mid] < target) left = mid + 1;
            else right = mid - 1;
          }
          return -1; // Target not found
        }
